# LexAnalytica Project Description

## Overview
LexAnalytica is an end-to-end AI-powered document intelligence system designed specifically for the legal domain. It automates the tedious tasks of document summarization, key entity identification, and cross-language translation, providing legal professionals with immediate, actionable insights. It now also supports scanned image documents through integrated OCR technology.

## Core Capabilities
- **Neural Summarization**: Employs state-of-the-art transformer models (**Legal-LED-16384**) to generate both concise extractive "Key Findings" and narrative abstractive "AI Synthesis" with a 16k token context window.
- **Institutional Entity NER**: Custom-tailored entity recognition to identify legal-specific markers such as Case IDs, Sections of Law, Organizations, and Judicial figures.
- **Full Bi-lingual Support**: Native Devanagari processing allows for the upload and analysis of Hindi legal documents with English output capabilities (and vice-versa).
- **OCR Document Support**: Integration of PaddleOCR enables extraction of text from scanned image files (PNG, JPG, JPEG), feeding them directly into the same intelligence pipeline as native documents.
- **Professional Reporting Engine**: Direct-to-client report generation in PDF format (with Unicode Devanagari support).

## Technical Foundation
- **Backend API**: FastAPI framework providing an asynchronous, low-latency interface for heavy NLP computations.
- **Reactive Dashboard**: A modern React-based UI focused on the user's workload, featuring predictive loading states, premium dark-mode aesthetics, and **Advanced Legal Entity Visualization** (using responsive wrapping pills for complex case markers).
- **Scalable NLP Pipeline**: A modular architecture separation between pre-processing, intelligence extraction, and document generation, allowing for localized model fine-tuning without breaking the interface.
- **OCR Layer**: PaddleOCR (PaddlePaddle) integrated as a transparent extraction layer within `nlp/extractor.py`, requiring zero changes to downstream summarization or entity logic.

## System Vision
LexAnalytica serves as a foundational "Legal Intelligence Layer," designed to bridge the gap between massive document repositories and rapid decision-making in high-stakes legal environments.
